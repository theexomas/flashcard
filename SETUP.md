# 한몽 카드 v2 — Database синк тохируулах заавар

Нийт 10 минутын ажил. Vercel-ийн тохиргоонд өөрчлөлт орохгүй — апп статик хэвээрээ, database нь Supabase дээр байрлана.

## 1. Supabase төсөл үүсгэх

1. supabase.com → өөрийн бүртгэлээрээ нэвтэрч **New project** үүсгэ
   (нэр: `hanmong-card`, region: **Northeast Asia (Seoul)** — Монгол, Солонгос хоёуланд ойр).
2. Төсөл үүссэний дараа зүүн цэснээс **SQL Editor** → New query → `supabase_setup.sql` файлын агуулгыг хуулж буулгаад **Run** дар. `user_data` хүснэгт RLS бодлоготойгоо үүснэ.

## 2. Auth тохиргоо

1. **Authentication → Sign In / Providers → Email** идэвхтэй эсэхийг шалга (default идэвхтэй).
2. Хялбар байлгах бол **Confirm email**-ийг унтраа — тэгвэл бүртгүүлмэгц шууд нэвтэрнэ. Асаалттай орхивол бүртгүүлсний дараа и-мэйлээ баталгаажуулах шаардлагатай (аль нь ч ажиллана).

## 3. Түлхүүрүүдээ index.html-д тавих

**Settings → API** хэсгээс хоёр утга хуулна:

- **Project URL** (ж: `https://abcdefgh.supabase.co`)
- **anon public** key

`index.html` дотор `<script>` хэсгийн эхэнд байгаа хоёр мөрийг соль:

```js
const SUPABASE_URL      = 'https://abcdefgh.supabase.co';
const SUPABASE_ANON_KEY = 'eyJhbGciOi...';
```

> anon key-г client код дотор задгай тавихад аюулгүй — энэ нь public түлхүүр бөгөөд өгөгдлийн хамгаалалтыг RLS бодлого хариуцна (хэрэглэгч бүр зөвхөн өөрийн мөрийг уншиж/бичнэ).

## 4. Deploy

Шинэ `index.html`-ээ Vercel дээрх төслөө солиод deploy хий (drag & drop эсвэл `vercel --prod`).

## 5. Ашиглах

1. Аппын баруун дээд буланд шинээр нэмэгдсэн **👤** товч дээр дар.
2. И-мэйл, нууц үгээрээ **Бүртгүүлэх** → **Нэвтрэх**.
3. 👤 товчны булан дахь жижиг цэг синкийн төлөвийг заана:
   - 🟢 ногоон — синк хийгдсэн
   - 🟡 шар — илгээж байна
   - 🔴 улаан — алдаа (интернэт эсвэл тохиргоо шалга)
4. Өөр төхөөрөмж дээр мөн адил нэвтрэхэд явц чинь автоматаар татагдана.

## Хэрхэн ажилладаг вэ

- Хариулт бүрийн дараа өөрчлөлт **1.5 секунд debounce**-тэйгээр Supabase руу илгээгдэнэ (хариулт бүрд биш, багцлаад). Таб хаагдах/нуугдахад мөн шууд илгээнэ.
- Нэвтрэх бүрд серверийн болон локал явцыг **merge** хийнэ: карт бүрээр аль их давтагдсан (reps өндөр, due хожуу) хувилбар нь үлдэнэ. Утсан дээрээ 50 карт, компьютер дээрээ өөр 30 карт давтсан бол хоёулаа хадгалагдана — юу ч дарагдаж устахгүй.
- Интернэтгүй үед localStorage дээрээ хэвийн ажиллаад, дараа нэвтрэхэд нэгтгэгдэнэ.
- Нэвтрээгүй ашиглавал өмнөх шигээ зөвхөн локал горимд ажиллана.
